# ClipLedger Support

For help with ClipLedger, use the GitHub issue tracker:

https://github.com/JasonHuang/ClipLedger/issues

## Before Opening an Issue

Please include:

- macOS version
- ClipLedger version
- A short description of the problem
- Steps to reproduce the issue
- Screenshots if they help explain the issue

Do not include sensitive clipboard contents in public issues.

## Privacy

ClipLedger stores clipboard history locally on your Mac and does not transmit clipboard data externally.

Privacy policy:

https://github.com/JasonHuang/ClipLedger/blob/master/docs/PRIVACY.md
